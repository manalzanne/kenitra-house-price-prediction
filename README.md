# Kenitra House Price Prediction

Machine Learning project to predict apartment prices in Kénitra using Linear Regression.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* FastAPI
* Joblib

## Project Steps

1. Data cleaning and preprocessing
2. Missing values and outlier handling
3. Feature engineering
4. Data scaling
5. Linear Regression
6. Model evaluation (MAE, R²)
7. Cross-validation
8. Model deployment with FastAPI

## Results

The model predicts apartment prices based on features such as surface area, number of rooms, building age, elevator availability, and neighborhood.
